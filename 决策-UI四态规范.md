# 决策-UI四态规范

## 决策内容

劳动仲裁模块（及风鸟平台各列表模块）须定义以下四种 UI 状态：

| 状态 | 触发条件 | 展示规范 |
|------|---------|---------|
| 加载中 | 接口请求中 | 骨架屏或 spinner |
| 请求失败 | 接口返回错误 | 错误提示文案 + 重试按钮 |
| 筛选无结果 | 筛选条件命中0条 | 「暂无符合条件的数据」+ 空状态图 |
| 空数据 | 企业本身无劳动仲裁记录 | 「暂无劳动仲裁数据」+ 空状态图 |

## 背景

PRD v4.11 全文未定义四态 UI，评审（R-005）发现缺失，
由评审人 test 确认接受，需在 §4.6 补充。
