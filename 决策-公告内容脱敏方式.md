# 决策-公告内容脱敏方式

## 决策内容

公告内容（非结构化文本）中人名脱敏的识别方式：

> **复用当事人名单（ent 表中 entity_type 为自然人的记录）对公告内容做字符串精确匹配替换，命中则替换为「姓+**」，未命中则保持原样。**

## 背景

PRD v4.11 §3.3 规定公告内容中人名需脱敏，但未说明如何识别非结构化文本中的人名。
评审（R-007）发现此缺失，由评审人 test 确认接受字符串匹配方案。
若后续需更高精度识别，可升级为 NLP 服务，届时更新本页。
