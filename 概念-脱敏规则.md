# 脱敏规则

## 定义

劳动仲裁模块中对自然人隐私信息的脱敏处理规则。

## 规则明细

| 字段场景 | 脱敏规则 | 示例 |
|---------|---------|------|
| 申请人/被申请人中的自然人 | 姓+** | 张三 → 张** |
| 申请人/被申请人中的企业 | 不脱敏 | 上海ABC公司 → 上海ABC公司 |
| 仲裁员 | 不脱敏 | 公务人员，信息公开 |
| 书记员 | 不脱敏 | 公务人员，信息公开 |
| 公告内容中的人名 | 姓+** | 与当事人规则一致 |
| 公告内容中的企业名 | 不脱敏 | 保持原样 |
