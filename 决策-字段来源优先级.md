# 决策-字段来源优先级

## 决策内容

当前端字段映射表中某字段来源为「ent表 或 主表」时，统一遵循以下优先级：

> **优先取 `ds_risk_labour_arbitration_ent` 表对应字段，ent 表值为空时降级取 `ds_risk_labour_arbitration` 主表对应字段。**

适用字段：
- `arbitration_org`（仲裁机构/发布机构）
- `case_reason`（案由）
- `publish_date`（发布日期）

## 背景

PRD v4.11 §2.2 字段映射表中多个字段写作「ent表.xxx 或 主表.xxx」，未说明优先级，
评审（R-003、R-004）发现此歧义，由评审人 test 确认接受上述规则。
