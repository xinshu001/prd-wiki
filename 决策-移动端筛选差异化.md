# 移动端筛选差异化

## 决策背景

移动端由于屏幕空间限制，筛选交互与 Web 端不同。

## 决策内容

- 移动端筛选维度：省份、年份、公告类型
- Web 端筛选维度：仲裁机构/发布机构、案由
- 移动端不提供搜索框，改用筛选组合替代

## 数据来源

- 省份：从 area_code 映射省级行政区划
- 年份：从 publish_date（送达）或 open_time（开庭）提取年份
- 公告类型：type 字段（1:送达, 2:开庭）
